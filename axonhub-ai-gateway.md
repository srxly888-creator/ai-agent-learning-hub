## 💥 核心亮点

> **AI 网关，让你无需改动一行代码即可切换模型供应商**

---

## 🎯 解决的痛点

### **当前问题**
1. **供应商锁定** - 换供应商需要改代码
2. **SDK 不兼容** - 不同 SDK 不同 API
3. **重构成本高** - 切换需要大量工作
4. **维护困难** - 多供应商代码复杂

---

## ✨ AxonHub 的价值

### **1. 无需改动代码**
- ✅ 无需重构
- ✅ 无需更换 SDK
- ✅ 只需更改配置

### **2. 透明转换**
> **透明地将你的请求转换为与任何支持的模型供应商兼容的格式**

- ✅ 自动格式转换
- ✅ 自动参数适配
- ✅ 自动错误处理

### **3. 多 SDK 支持**
- ✅ OpenAI SDK
- ✅ Anthropic SDK
- ✅ 任何 AI SDK

---

## 🔧 工作原理

```
你的代码（OpenAI SDK）
    ↓
AxonHub（透明转换）
    ↓
任意模型供应商
    ├─ OpenAI
    ├─ Anthropic
    ├─ Google
    ├─ 智谱 GLM
    └─ 其他...
```

### **1. 零代码改动**
```python
# Before: 只用 OpenAI
import openai
response = openai.ChatCompletion.create(...)

# After: 无需改动，只改配置
# 配置文件里切换供应商即可
```

### **2. 配置切换**
```yaml
# 从 OpenAI 切换到 GLM

```

- 透明转换
