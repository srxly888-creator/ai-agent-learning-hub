## 💥 核心痛点

> **Skills 越来越多管不过来？用 Claude Code、Cursor、Copilot、Codex……每个工具都有自己的 Skills 体系，装多了就乱了**

---

## 🎯 解决方案

**Skills Manager 把它们统一收到一个地方，集中管理、一键同步**

---

## ✨ 核心功能

### **1. 统一管理**
- ✅ 15+ AI 编程工具
- ✅ 一个地方管理所有 Skills
- ✅ 不用到处找

### **2. 一键同步**
- ✅ 一次配置，多处同步
- ✅ 自动同步到各工具
- ✅ 省时省力

### **3. 集中管理**
- ✅ 统一界面
- ✅ 清晰分类
- ✅ 易于维护

## 🚀 快速开始

### **1. 安装**
```bash
# 克隆项目
git clone https://github.com/xingkongliang/skills-manager.git

# 安装依赖
cd skills-manager
npm install

# 启动
npm start
```

### **2. 配置**
- 添加各工具的 Skills 目录
- 配置同步规则
- 设置自动同步

### **3. 使用**
- 统一管理界面
- 一键同步
- 集中维护
