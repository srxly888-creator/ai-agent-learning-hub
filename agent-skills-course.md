## 📖 课程内容

**Introduction to Agent Skills** - 官方课程，加起来不到 20 分钟

### **课程大纲**

1. **什么是 Skill**
   - 定义和核心概念
   - 与其他能力的区别

2. **怎么做第一个 Skill**
   - 入门步骤
   - 实践示例

3. **怎么做多文件和配置**
   - 进阶技巧
   - 配置管理

4. **和其他能力有什么区别**
   - Skills vs Plugins
   - Skills vs MCP
   - Skills vs Tools

5. **怎么分享**
   - 发布流程
   - 社区分享

## 💡 关键洞察

### **大多数人忽略的东西**

❌ **错误的做法**：
- 只刷案例
- 不看能力是怎么构建的
- 忽略基础概念

✅ **正确的做法**：
- 理解能力结构
- 学习基础原理
- 从底层掌握构建方法
