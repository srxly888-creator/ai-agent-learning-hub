# Prompt Engineering

> 📝 来源：[ai-knowledge-graph](https://github.com/nxs9bg24js-tech/ai-knowledge-graph) — YouTube 字幕整理笔记

## 定义
通过设计和优化提示词（Prompt），引导大语言模型生成高质量输出的技术与方法。

---

## 核心要点

### 1. 基础技巧
- **清晰具体**: 明确任务目标和输出格式
- **角色扮演**: 赋予模型特定角色（专家、助手）
- **示例驱动**: 提供输入输出示例（Few-shot）
- **结构化提示**: 使用分隔符、列表、表格

### 2. 高级技巧
- **思维链 (CoT)**: 引导模型逐步推理
- **思维树 (ToT)**: 探索多条推理路径
- **自我反思**: 让模型评估自己的输出
- **迭代优化**: 多轮对话逐步完善

### 3. 常见模式
- **RAG 模式**: 检索 + 生成
- **Agent 模式**: 规划 + 执行
- **工具调用模式**: 感知 + 决策 + 执行
- **多轮对话模式**: 上下文 + 迭代
