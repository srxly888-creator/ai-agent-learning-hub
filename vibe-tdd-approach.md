## 🔄 TDD 工作流

### **传统 TDD**
```
写测试 → 写代码 → 测试通过 → 重构
```

### **Vibe TDD**
```
自然语言描述测试 → Vibe Coding 实现 → 自动测试 → 快速迭代
```
