## 🎯 真正的挑战

> **发现 skill 是第一步，养活一个不断膨胀的知识库才是长期工程**

### **核心问题**

1. **怎么淘汰？**
   - 过时的 skill 如何识别
   - 低质量的 skill 如何过滤
   - 冗余的 skill 如何合并

2. **怎么合并？**
   - 相似 skill 的合并策略
   - 冲突 skill 的解决机制
   - 统一的命名规范

3. **怎么防止互相矛盾？**
   - 检测 skill 之间的冲突
   - 解决矛盾的优先级
   - 保持知识库的一致性
