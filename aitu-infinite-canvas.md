## 💥 核心亮点

> **把 AI 生图、AI 生视频和一整套白板工具合在一个无限画布上**

---

## 🎯 解决的痛点

### **大多数 AI 绘图工具的问题**
- ❌ 只管生成
- ❌ 画完就没了
- ❌ 需要来回导出导入
- ❌ 不方便编排

### **AITU 的解决方案**
- ✅ 无限画布
- ✅ 生成内容直接编排
- ✅ 不用导出导入
- ✅ 一站式工作流

---

## ✨ 核心功能

### **1. AI 生图**
- AI 生成图片
- 直接在画布中显示
- 可继续编辑

### **2. AI 生视频**
- AI 生成视频
- 无限画布中播放
- 与图片协同

### **3. 白板工具**
- 思维导图
- 流程图
- 手绘功能

### **4. 无限画布**
- 无限空间
- 自由编排
- 缩放自如

---

## 🔄 工作流对比

### **传统方式**
```
AI 生图工具 → 导出 → 白板工具 → 导入 → 编排
    ↓
AI 生视频工具 → 导出 → 白板工具 → 导入 → 编排
    ↓
来回切换，效率低
```

### **AITU 方式**
```
AITU 无限画布
    ├─ AI 生图（直接生成）
    ├─ AI 生视频（直接生成）
    ├─ 思维导图
    ├─ 流程图
    └─ 手绘
    ↓
一站式编排，效率高
```

## 🚀 快速开始

### **1. 安装**
```bash
# 克隆项目
git clone https://github.com/ljquan/aitu.git

# 安装依赖
cd aitu
npm install

# 启动
npm start
```

### **2. 使用**
- 打开无限画布
- 选择 AI 生图/生视频
- 生成内容
- 在画布中编排
