## 📚 MSA 论文（Memory Sparse Attention）

### **来源**
- **作者**: 陈天桥（中国前首富）旗下团队 @EverMind
- **推文**: https://x.com/gosailglobal/status/2035033497405174023
- **论文**: Memory Sparse Attention（MSA）

### **核心创新**

**一句话说清楚它是什么**：
> 让大模型原生拥有超长记忆。不是外挂检索，不是暴力扩窗口，而是把「记忆」直接长进了注意力机制里，端到端训练。

### **过去方案的问题**

| 方案 | 问题 |
|------|------|
| **RAG** | 检索不准确，丢失上下文 |
| **暴力扩窗口** | 计算成本高，效果差 |
| **外挂检索** | 不是端到端，不够智能 |

### **MSA 的优势**

- ✅ **原生支持** - 记忆直接长进注意力机制
- ✅ **端到端训练** - 无需外挂检索
- ✅ **超长记忆** - 支持超长上下文
- ✅ **高效计算** - 稀疏注意力机制
